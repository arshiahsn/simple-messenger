<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/arshiahsn/simple-messenger.git
   ```
2. Download and Install NodeJS
3. Run NPM install
   ```sh
   npm install
   ```
   
### Running


  ```sh
  npm run start
  ```
  
### Usage
Open your browser and navigate to:

  ```sh
  localhost:3000/
  ```
